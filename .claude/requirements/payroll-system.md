# 사용자 원본 요구사항: 급여/인사 시스템

## 요구사항 (원문)
1. 포괄임금 계산 : 연장/야간/휴일수당 계산
2. 급여명세서 교부
3. 퇴직금 계산
4. 4대보험 신고 및 관리
5. 근로계약서 작성
6. 고용지원금 매칭

## 사용자 결정사항
- 전체 임금체계 동시 구현 (포괄/통상/시급/일급/연봉)
- 소득세 + 지방소득세 포함
- 비과세 수당 분리 관리 (식대 20만, 교통비 20만 한도)
- 고용지원금: 외부 API 연동
- UI 위치: 별도 메뉴 분리

## 검증 기준
- 포괄임금: 약정 시간 개념 적용, 초과분만 추가 수당. 단순 시급제 추가수당이 아님.
- 고용지원금: 외부 API(고용24 등) 실제 연동. 내부 매칭만은 불충분.
- 비과세: 과세분에만 소득세+4대보험 적용. 비과세 항목별 한도 관리.
- SSOT: Admin 설정 → 급여 계산 → 명세서 → Employee 조회가 동일 데이터.
- RAG: 변경 시 관련 RAG 파일 업데이트.
